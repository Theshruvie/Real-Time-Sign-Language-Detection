About the Project

This Sign Language Interpreter uses deep learning to recognize and translate hand gestures into text in real-time. The system employs computer vision techniques with OpenCV for hand detection and segmentation, followed by a Convolutional Neural Network built with TensorFlow/Keras to classify the detected gestures.

The project aims to bridge communication gaps between deaf and hearing individuals by providing a tool that can interpret sign language in real-time, making everyday interactions more accessible.
